# Reports

This directory stores human-readable reports for each probing group.

Use one report per experiment group:

```text
exp_1_sd_tiny_target_retention.md
exp_2_sd_cross_attention_response.md
exp_3_sd_self_attention_structure.md
exp_4_full_vs_crop.md
```

Each report should link to the exact config and result directory used.

The attention-derived spatial prior is a deferred stage after these four
foundation experiments.
